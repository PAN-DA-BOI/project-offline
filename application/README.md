# PC application
the PC app allows messaging through a PC, there are packages for linux, windows. both with and without a GUI. all packages can be found in "releases"

### installation
first get the package for your operating system. 

packages are clearly labeled as suck:
> operatingSystem-version-guiChoice.zip

for example a windows program without GUI will look like this
> Windows-v1.0.2-NoGUI.zip

simply extract, and run the program
