# Project Offline
offline, decentralised communication that doesnt piggyback off of any other infrastructure is important to have in case of emergency, or anytime you need to communicate in a way that wont have others listening in.

### why not meshtastic?
meshtastic has out grown from something that we could just add or intigrate quickly. we also wanted server side intigrations so that communication is never tied to a single device. meshtastic is of course a great option, but it simply doesnt fit our needs.

## getting started
first, figure out what device your installing firmware or software too, then go to the directory with that name

for example, if your installing firmware to a T-Deck Pro, go to that directory and follow the instructions there.
