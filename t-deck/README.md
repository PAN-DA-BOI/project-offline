# T-Deck Pro
the T-Deck Pro is a lora enabled "all in one" package for communication. it has display, keyboard, GPS, SD card slot, and even touchscreen, all in one package, like an old school blackberry.  perfect for communication on the go

### About
this is all based off the example code in [Xinyuan-LilyGO/T-Deck-Pro](https://github.com/Xinyuan-LilyGO/T-Deck-Pro/tree/master). this allows for communication across multiple hardware methods. 
## setup
all packages can be found in "releases". packages are clearly labeled as such:
> TDECK-version.zip

for example a T-deck will look like this

> TDECK-v1.0.2.bin

to flash the bin using the same instructions given in the official [T-Deck-Pro](https://github.com/Xinyuan-LilyGO/T-Deck-Pro/tree/master/firmware) repository
