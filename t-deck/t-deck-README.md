
# T-Deck Pro
the T-Deck Pro is a lora enabled "all in one" package for communication. it has display, keyboard, GPS, SD card slot, and even touchscreen, all in one package, like an old school blackberry.  perfect for communication on the go

### Info
this is all based off the example code in [Xinyuan-LilyGO/T-Deck-Pro](https://github.com/Xinyuan-LilyGO/T-Deck-Pro/tree/master) 
## getting a server setup
```
git clone https://github.com/PAN-DA-BOI/project-offline.git
chmod +x .project-offline/t-deck/setup.sh
.project-offline/t-deck/setup.sh
```
